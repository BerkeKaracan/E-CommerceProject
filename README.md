# Market Dashboard

A modern e-commerce dashboard featuring a responsive grid layout and independently scrollable sections.

## 🚀 Tech Stack

### Frontend

- **Framework:** [Next.js](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)

### Backend

- **Framework:** [FastAPI](https://fastapi.tiangolo.com/)
- **Database:** [PostgreSQL](https://www.postgresql.org/)

## ✨ Features

- **Responsive Architecture:** \* Mobile: Vertical stacking layout.
  - Desktop: Multi-column product grid with a fixed right sidebar for the cart preview.
- **Layout Mechanics:** Fixed `h-screen` viewport with separately scrollable product and cart areas to prevent global page scrolling.
- **UI Details:** Restricted text-selection behavior for a cleaner interface and customized hover states for navigation components.

## 🛠️ Getting Started

### Prerequisites

- Node.js 18+
- Python 3.10+
- PostgreSQL

### Running the Frontend Locally

1. Clone the repository.
2. Install dependencies:

   npm install

3. Start the development server:

   npm run dev

4. Open http://localhost:3000 with your browser to see the result.
