# 🛍️ Premium E-Commerce Platform

[![Deployed on Vercel](https://img.shields.io/badge/Live_Demo-Vercel-black?logo=vercel&style=for-the-badge)](https://e-commerce-project-market.vercel.app/)

A feature-rich, high-performance modern e-commerce frontend built to demonstrate advanced web development practices. This project features a fully responsive design, seamless dark mode integration, complex state management, and a dynamic loyalty points system.

**🔴 Live Demo:** [View the project live on Vercel](https://e-commerce-project-market.vercel.app/)

⚠️ **Disclaimer:** This is a portfolio project. No real transactions occur, and no real credit card data is processed.

## ✨ Key Features

- 🌓 **Absolute Dark/Light Mode:** Flawless transition across all components, including modals, dropdowns, and loading spinners.
- 🎁 **Dynamic Loyalty Program:** Automated points calculation based on order history. Users can redeem 50 points for a functional $5 discount code (LOYALTY5).
- 🛒 **Advanced Cart & Checkout:** Real-time price calculation, dynamic subtotaling, shipping costs, and promo code validation.
- 👤 **Comprehensive User Profile:** Accordion-style detailed order history, real-time settings, saved items, and product review displays.
- 🤖 **AI Assistant Interface:** Integrated chat UI simulating an AI-powered customer support and shopping assistant.
- 🔍 **Smart Product Discovery:** Client-side search, category filtering, and sorting (Price, Newest, Recommended).
- 📱 **Mobile-First Design:** Fluid responsiveness utilizing bottom sheets, hamburger menus, and touch-friendly UI elements.

## 🛠️ Tech Stack

- **Framework:** Next.js (React)
- **Styling:** Tailwind CSS v4
- **Language:** TypeScript
- **State Management:** React Context API & Custom Hooks
- **Architecture:** Component-based, modular structure

## 🚀 Getting Started

To run this project locally, follow these steps:

1. Clone the repository:
   git clone [https://github.com/BerkeKaracan/E-CommerceProject.git](https://github.com/BerkeKaracan/E-CommerceProject.git)

2. Navigate to the project directory:
   cd e-commerceproject

3. Install dependencies:
   npm install

4. Run the development server:
   npm run dev

5. Open http://localhost:3000 with your browser to see the result.

## 🎯 Project Highlights

- **Anti-Aliasing Fixes:** Handled subpixel rendering artifacts in webkit browsers for perfect circular loading spinners.
- **Z-Index Standardization:** Carefully managed z-index hierarchy (z-40, z-50) to ensure modals, drop-downs, and navigation bars never conflict.
- **Form Validations:** Simulated authentication (AuthModal) with email formatting checks, password length requirements, and error handling.
